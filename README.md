# Code for manuscript figures

This repository contains R scripts used to generate figures associated with the manuscript.

Files

* PCA_analysis_06.R

  * PCA analysis and generation of Figure 5A

* Upset_05.R

  * UpSet analysis and generation of Figure 5B

Required R packages:

* readxl
* dplyr
* ggplot2
* ggpubr
* stringr
* tidyr
* ggrepel
* ComplexUpset
